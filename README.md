# search-gitrepo


**How to start the Server** 


To Install dependencies 

```sh

cd server

yarn 

```

Start the server

```sh

yarn dev

```

or 

```sh

yarn start 

```

To see the API docs

```sh

yarn docs 

```


**How to start the client** 


Install dependencies 

```sh
cd client

yarn

```

To start the react client app.

```sh

yarn start

```
