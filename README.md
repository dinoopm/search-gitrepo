# search-gitrepo


**How to start the Server** 


To Install dependencies 

```sh

cd server

yarn 

```

Start the server

```sh

yarn dev

```

or 

```sh

yarn start 

```


**How to start the client** 


Install dependencies 

```sh
cd client

yarn

```

or

```sh
npm install

```

To start the react client app.

```sh

yarn start

```
